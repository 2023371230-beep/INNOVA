# INNOVA® Trade & Consulting Group

Sitio web estático para comercialización industrial, consultoría y capacitación.

## Estructura del proyecto

- `index.html` - Página principal
- `comercializacion.html` - Servicios de comercialización
- `capacitacion.html` - Capacitación y consultoría
- `contact.html` - Página de contacto
- `products.html` - Catálogo de productos
- `static/` - CSS, JavaScript y assets
- `Imagenes/` - Imágenes del sitio

## Despliegue en GitHub Pages

1. Asegúrate de que todos los cambios estén guardados
2. Sube el repositorio a GitHub
3. Ve a Settings → Pages
4. Selecciona la rama `master` y carpeta `/ (root)`
5. Guarda y espera unos minutos

Tu sitio estará disponible en: `https://[tu-usuario].github.io/[nombre-repo]`
